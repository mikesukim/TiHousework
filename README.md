## Getting Started / 어떻게 시작하나요?
### Prerequisites / 선행 조건
- npm
- react-native
- Xcode 12.5 version

### Installation / 설치
```
git clone
npm install
cd ios && pod install

* credentials폴더 를 src에 포함시켜함!
credentials폴더를 얻으려면 성언한테 연락하세요!
```
### Running / 실행
run project on simulator
```
npm run ios
혹은
npm run android
```
